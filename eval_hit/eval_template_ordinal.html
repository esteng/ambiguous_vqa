Image Question Answering

Welcome! Thank you for participating in this task. Please read the instructions carefully.

You will be presented with an image paired with a question about the image and an answer to the question.

Your task is to judge whether the given answer is a valid answer to the question. The options are yes, no, and maybe; maybe can be used if you are not sure whether it is valid or not.

You can click the buttons or use the number keys (1,2,3) to select options. Note that you may have to click the HIT once to get the numbers to work.

If you have any questions or comments, please use the comment box, or send an e-mail to textual.choreography@gmail.com

Otherwise, please leave the comment box blank.
